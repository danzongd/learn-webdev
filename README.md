# Aequilum | Digital Sanctuary (数字圣所) 🏛️

*"Je me révolte, donc nous sommes." (I rebel—therefore we exist. — Albert Camus)*

Welcome to my personal *portfolio* (作品集) and *digital archives* (数字档案馆). This project is an exploration of *minimalism* (极简主义), classical *aesthetics* (美学), and pure *code* (代码). It serves as a quiet space to document my journey through computer science, philology, and philosophy.

## 🛠️ Tech Stack (技术栈)

Driven by an obsession with *purity* (纯粹) and *performance* (性能), this website is built with absolute zero JavaScript. Everything from the layout to the complex interactions is handled purely by CSS.

* **HTML5**: Semantic and accessible *structure* (结构).
* **Tailwind CSS v4**: Utility-first *styling* (样式) via `@import`.
* **daisyUI**: Utilizing the minimalist `lofi` *theme* (主题) for a classical, editorial *elegance* (优雅).
* **Custom CSS**: Extremely smooth *animations* (动画) driven by customized Bézier curves (贝塞尔曲线), intricate `clip-path` text reveals, and a subtle *film grain* (胶片颗粒) overlay for an analog *texture* (质感).

## 🗂️ Architecture (架构)

The *repository* (仓库) is structured as a multi-page *ecosystem* (生态系统):

* `index.html` - **L'Accueil (主页)**: The personal *manifesto* (宣言) and an overview of my core intellectual domains (Logic, Philology, Philosophy).
* `archives.html` - **Les Archives (档案馆)**: A curated *reading ledger* (阅读清单) covering Constitutional Law texts, Classical Linguistics, and Pure Sciences.
* `galerie.html` - **La Galerie (画廊)**: A pure CSS *Masonry Grid* (瀑布流网格) showcasing photography, featuring a custom grayscale-to-color hover *interaction* (交互).
* `contact.html` - **Correspondance (通讯录)**: A minimalist, typography-driven *directory* (目录) for digital connections.
* `src/style.css` - The core *stylesheet* (样式表) housing custom *keyframes* (关键帧) and premium typography (Playfair Display & EB Garamond).

## ⚙️ Development (开发)

Since the project uses Vite and Tailwind v4 for rapid building, local *development* (开发) is highly straightforward:

1. Clone the *repository* (仓库) to your local machine.
2. Install the necessary *dependencies* (依赖): `npm install`
3. Start the local development *server* (服务器): `npm run dev`

Alternatively, because the final build relies entirely on static HTML and CSS, you can simply open any `.html` file directly in a modern *browser* (浏览器) to view the content.
